# Digital Certificate Verification  
**digital-cert-verify**

Digital Certificate Verification adalah aplikasi **berbasis web menggunakan Laravel** untuk memverifikasi keaslian sertifikat digital (*Certificate of Completion*) dengan memanfaatkan **QR Code** dan **Digital Signature (ECDSA)**.  
Sistem ini dirancang untuk mencegah pemalsuan sertifikat serta mempermudah proses validasi secara cepat, aman, dan akurat.

---

## 🚀 Fitur Utama
- 🔐 **Digital Signature (ECDSA)** untuk menjamin keaslian sertifikat
- 📷 **QR Code Verification** langsung melalui browser
- 🌐 **Web Application (Laravel)** tanpa instalasi aplikasi tambahan
- 🧾 Validasi data sertifikat secara real-time
- ❌ Deteksi perubahan data atau sertifikat palsu
- 📊 Manajemen data sertifikat (CRUD)

---

## 🛠️ Teknologi yang Digunakan
- **Framework**: Laravel
- **Bahasa Pemrograman**: PHP
- **Database**: MySQL 
- **Cryptography**:  
  - ECDSA (Elliptic Curve Digital Signature Algorithm)
- **QR Code**:  
  - QR Code Generator & Scanner
- **Frontend**:  
  - Blade Template, HTML, CSS, JQuery
